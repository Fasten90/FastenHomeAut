Release Notes for

STM32Cube Expansion of Class B library

Copyright 2015 STMicroelectronics

 

Update History

V2.0.0 / 14-December-2015

Main Changes
  • First official release of the FW certified by UL certification authority
  • License disclaimer update
  • Correction after MISRA check
  • Use of latest HAL drivers

V2.0.0RC5 / 11-November-2015

Main Changes
  • HSI option is implemented additionally
  • IWDG is started prior its initialization

V2.0.0RC4 / 17-August-2015

Main Changes
  • RAM tests are replaced by procedures written in assembler
  • IAR CRC calculation issue is compensated (HAL driver is replaced by direct control of CRC unit)

V2.0.0RC2 / 8-December-2014

Main Changes
  • Added support of STM32L1xx HAL drivers

V2.0.0RC1 / 11-June-2014

Main Changes
  • First release of common self-test stack supported by HAL drivers
